#  Code Explainer AI

An AI-powered tool that explains any code snippet in plain English.

# Features
- Supports Python, JavaScript, Java, C++, SQL
- Line-by-line explanation
- Identifies key concepts & patterns
- Suggests improvements

# Built With
- Python
- Streamlit (UI)
- Groq API / LLaMA 3.3 (LLM)

# How to Run
pip install streamlit groq
streamlit run app.py
